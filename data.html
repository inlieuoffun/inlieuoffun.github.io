The following data about In Lieu of Fun are exported as JSON to support programmatic use:

  • The Episode Log:
    {{ site.url }}/episodes.json
    • The episode record for the latest published or upcoming episode:
      {{ site.url }}/latest.json
    • The episode record for a specific episode (for example, 180):
      {{ site.url }}/episode/180.json
  • The Guest List:
    {{ site.url }}/guests.json
  • The data schema:
    {{ site.url }}/schema.json
  • An inverted index of terms extracted from episode summaries and details:
    {{ site.url }}/textindex.json

    The index contains two entries, "terms" and "stops". Each term maps a single word stem to the words based on that stem, and for each word the episodes that contain it. Words are normalized to lower-case, Tags (e.g., has:links) are recorded in the same format as words.

    The `"stops"` are an array of stopwords excluded from indexing. This can be used to filter query terms.

    The stems in the terms index are created using the Porter 2 stemming algorithm.

These data are generated statically when the site is updated. Tools can use the If-None-Match HTTP header to poll for changes.
